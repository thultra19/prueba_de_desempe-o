#  Student registry system

This console python program is desinged to manage the student list of an educational institution.
the full program can be found [here](https://github.com/thultra19/prueba_de_desempe-o.git/)

---

## 🚀 Funcionality

- Add, show, search, update and delete students from the system  
- save and load form CSV type Files 
- interactive console menu  

---

## ⚙️ Software execution

python Main.py

---

## 🧱 Structure

- Main.py       # Main menu
- Funtions.py   # File management

---
